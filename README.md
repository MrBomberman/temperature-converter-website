It is my converter of celsiuis to fahrenheit and fahrenheit to celsius.
Created on next js. For styles I used css modules as it is a very good technology
## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
